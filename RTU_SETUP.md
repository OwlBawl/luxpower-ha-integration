# Modbus RTU Setup Guide

This guide explains how to connect your LuxPower inverter directly to Home Assistant using an RS-485 to USB converter (Modbus RTU protocol).

## Hardware Requirements

### RS-485 to USB Converters (Tested/Compatible)
- **FTDI FT232** based converters
- **Waveshare USB to RS485** converter
- **CH340** based converters
- Any standard USB to RS-485 adapter supporting 3.3V/5V logic levels
